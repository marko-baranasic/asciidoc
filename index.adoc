== Sažetak
Ovaj dokument objašnjava **osnovne principe AsciiDoc** sintakse. Namijenjen je studentima koji već poznaju Markdown, ali žele naučiti alat koji nudi više mogućnosti za strukturiranje i oblikovanje akademskog i tehničkog teksta.

== Što je AsciiDoc
AsciiDoc je *plain-text* format za pisanje strukturiranih dokumenata. Tekst je lako čitljiv u izvornom obliku i može se pretvoriti u *HTML*, *PDF* ili *DOCX* pomoću alata *Asciidoctor*.

=== Glavne prednosti u odnosu na Markdown
- podržava naslove svih razina (`=`, `==`, `===`)
- automatski izrađuje sadržaj i numerira poglavlja
- koristi *atribute i varijable*
- jednostavno dodaje__ fusnote__, __tablice__, __citate__ i __blokove koda__

== Struktura dokumenta
Na početku se obično nalazi naslov, autor, datum i osnovni atributi.
Naslovi se označavaju simbolima jednakosti (`=`).
Više znakova znači nižu razinu naslova.

```adoc
Autor <email@domena.com>
:revdate: 2025-11-06
:doctype: article
:toc: 
:sectnums:
```


== Formatiranje teksta
Tekst se može podebljati, nagnuti ili precrtati.

```adoc
*podebljano*, _kurziv_, [line-throught] #precrtano#
```



*podebljano*, _kurziv_, [line-throught] #precrtano#

== Popisi
Numerirani popisi koriste brojeve ili točke.
Nenumerirani popisi koriste zvjezdice ili crtice.



```adoc
.numerirani popis
.. Stavka 1
... Stavka 2
```
.. Stavka 1
... Stavka 2


```adoc
*nenumerirani popis
** Stavka 1
*** Stavka 2
```
** Stavka 1
*** Stavka 2

== Citati i fusnote

Citati se prikazuju u posebnom bloku, uz ime autora. footnote:[Navodi se izjave važnih osoba.]

[quote, Alan Kay]
____
Najbolji način da predvidiš budučnost je da je izmiliš
____

Fusnote se dodaju na kraju rečenice i sadrže kratko objašnjenje. footnote:[ovo je objašnjenje predhodnog teksta]

== Tablice
Tablice omogućuju pregledno prikazivanje podataka u redovima i stupcima.
Svaka ćelija odvaja se vertikalnom crtom (|).

[cols="1,2", options="header"]
|===
| Element | Opis
| Naslov | Definira glavni naslov dokumenta
|`:toc:` |Dodaje sadržaj na početku dokumnta
| `:sectnums:` | Numerira poglavlja automatski
|===

== Kod
Blokovi koda koriste se za prikaz primjera u programskim jezicima.
Mogu imati i oznaku jezika, npr. python ili bash.

[source, python]
----
def pozdrav (ime:):
    print(f"Pozdrav, {ime}!")

pozdrav("studenti")

----

== Napomene
Posebni blokovi označavaju važne dijelove teksta, poput upozorenja ili savjeta.
Primjeri su: TIP, NOTE, WARNING i IMPORTANT.

[TIP]
====
Koristite Ssciidoc kada želite pisat profesionalno uređenu dokumentaciju.
====

[WARNING]
====
Pripazitr da se u istom dokumentu ne mijesa `Mardown` i `asciidoc`
====
